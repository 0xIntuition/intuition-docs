# GraphQL API Overview

Intuition provides a GraphQL API as an easy and flexible way to query its knowledge graph.

If this is your first time encountering GraphQL, you can read more at [graphql.org](https://graphql.org/).

Until mainnet launch, the Intuition team hosts an open GraphQL API powered by Hasura that you can query at [https://testnet.intuition.sh/v1/graphql](https://testnet.intuition.sh/v1/graphql).
If you want to learn more or run the backend yourself, see [Run an Intuition node](/docs/run-node/run-an-intuition-node).

To leverage this GraphQL API, there are a few ways to get started depending on the level of abstraction and customizability you are looking for:
