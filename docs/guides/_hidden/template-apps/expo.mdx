# Expo App Template

This guide will help you get started with building a cross-platform mobile application using Expo and Intuition. The template provides a solid foundation for building iOS and Android apps with React Native.

## Features

- Expo SDK 50
- Intuition SDK integration
- Navigation with Expo Router
- Authentication flow
- Secure storage
- UI with NativeWind
- TypeScript support
- Testing with Jest
- Offline support
- Deep linking

## Prerequisites

Before you begin, ensure you have:
- Node.js 18 or later installed
- npm or yarn package manager
- Expo CLI installed (`npm install -g expo-cli`)
- iOS Simulator or Android Emulator (optional)
- An Intuition account

## Getting Started

1. Create a new project using the template:

```bash
npx create-intuition-app my-expo-app --template expo
cd my-expo-app
```

2. Install dependencies:

```bash
npm install
```

3. Set up your environment variables:

```bash
cp .env.example .env
```

Edit the `.env` file with your configuration values.

4. Start the development server:

```bash
npm start
```

This will open the Expo development tools where you can run the app on iOS, Android, or web.
