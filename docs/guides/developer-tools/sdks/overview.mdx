## âš¡ Quick Start

Here's how to initialize the SDK in one step:

```typescript
import { setup } from '@0xintuition/sdk';

const { client, protocol } = await setup({
  token: YOUR_GRAPHQL_TOKEN,
  web3Provider: window.ethereum,
  network: 'mainnet',
});
```

This gives you:

- **client**: for GraphQL-based reads and writes (e.g. querying vaults or creating atoms)
- **protocol**: for smart contract calls (e.g. deposits, redemptions)

This deposits ETH into the atom vault, mints vault shares for the user, and enforces any curve logic associated.



## ğŸ’» Real Example: React Component

Here's a self-contained UI snippet that previews and executes a deposit:

```tsx
import React from 'react';
import { setup } from '@0xintuition/sdk';

function DepositComponent({ atomId }) {
  const [shares, setShares] = React.useState(null);

  React.useEffect(() => {
    (async () => {
      const { client } = await setup({ 
        token: TOKEN, 
        web3Provider: window.ethereum, 
        network: 'mainnet' 
      });
      const preview = await client.previewDeposit({ 
        atomId, 
        amountWei: '1000000000000000000', 
        curveId: 4 
      });
      setShares(preview.shares);
    })();
  }, [atomId]);

  const handleDeposit = async () => {
    const { protocol } = await setup({ 
      token: TOKEN, 
      web3Provider: window.ethereum, 
      network: 'mainnet' 
    });
    const tx = await protocol.vaults.depositAtomCurve(atomId, '1000000000000000000', 4);
    await tx.wait();
    alert('âœ… Deposit confirmed');
  };

  return (
    <div>
      <p>Estimated Shares: {shares}</p>
      <button onClick={handleDeposit}>Deposit 1 ETH</button>
    </div>
  );
}
```
