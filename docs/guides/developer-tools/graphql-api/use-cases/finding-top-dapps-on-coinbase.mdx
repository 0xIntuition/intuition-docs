# Finding Top Dapps on Coinbase

This guide shows you how to progressively build queries to discover and rank the top decentralized applications (Dapps) on Coinbase by market cap.

Follow these 4 queries step-by-step. Each one builds on the previous to reach the final result: **a ranked list of top Dapps on Coinbase**.

## Understanding the Progression

- **Step 1**: Basic query to see all organizations (Dapps) in the system
- **Step 2**: Same query but filtered to only show Coinbase-related Dapps (notice how results narrow down)
- **Step 3**: Add performance metrics from vaults (market cap, shares, positions) - data is now richer but not yet sorted
- **Step 4**: Same as Step 3 but now **sorted by market cap** to show the top performing Dapps first!
