# Remix App Template

This guide will help you get started with building a full-stack web application using Remix and Intuition. The template provides a solid foundation with best practices and common functionality already set up.

## Features

- Full-stack Remix application structure
- Intuition SDK integration
- Authentication setup
- Database integration
- Styling with Tailwind CSS
- TypeScript support
- Testing setup with Vitest
- Docker configuration

## Prerequisites

Before you begin, ensure you have:
- Node.js 18 or later installed
- npm or yarn package manager
- Basic knowledge of Remix and React
- An Intuition account

## Getting Started

1. Create a new project using the template:

```bash
npx create-intuition-app my-remix-app --template remix
cd my-remix-app
```

2. Install dependencies:

```bash
npm install
```

3. Set up your environment variables:

```bash
cp .env.example .env
```

Edit the `.env` file with your configuration values.

4. Start the development server:

```bash
npm run dev
```

Your app will be running at `http://localhost:3000`.
