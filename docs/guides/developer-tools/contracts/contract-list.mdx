## Contract Deployment

### Deployment Scripts

```javascript
// Example deployment script
const { ethers } = require("hardhat");

async function main() {
  const [deployer] = await ethers.getSigners();
  console.log("Deploying contracts with account:", deployer.address);

  // Deploy Trust Bonding Contract
  const TrustBonding = await ethers.getContractFactory("TrustBonding");
  const trustBonding = await TrustBonding.deploy();
  await trustBonding.deployed();
  console.log("TrustBonding deployed to:", trustBonding.address);

  // Deploy Bonding Curve Contract
  const BondingCurve = await ethers.getContractFactory("BondingCurve");
  const bondingCurve = await BondingCurve.deploy();
  await bondingCurve.deployed();
  console.log("BondingCurve deployed to:", bondingCurve.address);

  // Deploy Multi-Vault Contract
  const MultiVault = await ethers.getContractFactory("MultiVault");
  const multiVault = await MultiVault.deploy();
  await multiVault.deployed();
  console.log("MultiVault deployed to:", multiVault.address);
}

main()
  .then(() => process.exit(0))
  .catch((error) => {
    console.error(error);
    process.exit(1);
  });
```

## Contract Monitoring

### Event Monitoring

Monitor contract events for important activities:

```javascript
// Monitor bond creation events
trustBonding.on("BondCreated", (bondId, creator, beneficiary, amount) => {
  console.log(`New bond created: ${bondId} by ${creator} for ${beneficiary}`);
});

// Monitor bonding curve purchases
bondingCurve.on("TokensPurchased", (buyer, amount, cost) => {
  console.log(`Tokens purchased: ${amount} by ${buyer} for ${cost} ETH`);
});
```

### Health Checks

Regular health checks for contract functionality:

```javascript
async function performHealthCheck() {
  // Check contract balances
  const trustBondingBalance = await ethers.provider.getBalance(trustBonding.address);
  const bondingCurveBalance = await ethers.provider.getBalance(bondingCurve.address);
  
  console.log("Contract Balances:");
  console.log(`Trust Bonding: ${ethers.utils.formatEther(trustBondingBalance)} ETH`);
  console.log(`Bonding Curve: ${ethers.utils.formatEther(bondingCurveBalance)} ETH`);
}
```
